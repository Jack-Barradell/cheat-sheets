# Target Recon



## Host Discovery 



### Nmap

#### No Port Scan

    nmap -sn [IP][CIDR]

##### Params 

    [IP]: IP address to test, can scan a range by providing a CIDR
    [CIDR] (optional): Shorthand method of wriging a subnet mask, commonly /24 or /16

##### Examples

    nmap -sn 192.168.212.25

    nmap -sn 192.168.56.0/24

    nmap -sn 192.168.0.0/16
